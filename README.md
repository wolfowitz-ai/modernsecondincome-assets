# modernsecondincome-assets

Asset storage for [ModernSecondIncome](https://modernsecondincome.com).

## Structure

```
pdfs/
  side-business-automation-stack.pdf   ← MSI lead magnet (make-com-2026-03)
```

## PDF Download URL

Once the file is committed, the stable public download URL is:

```
https://github.com/wolfowitz-ai/modernsecondincome-assets/raw/main/pdfs/side-business-automation-stack.pdf
```

## Adding a New PDF

1. Replace or update the file at `pdfs/side-business-automation-stack.pdf`
2. Commit and push to `main`
3. The download URL above remains stable — no link updates needed anywhere

## Notes

- Use `raw.githubusercontent.com` or the `/raw/main/` GitHub path for direct download links
- GitHub Releases are an option for versioned PDFs if needed in future
